# starlink-mon
Docker Compose and Grafana configs to stand up:
* [starlink-grpc-tools](https://github.com/sparky8512/starlink-grpc-tools)
* InfluxDB
* Grafana
  * Starlink Dashboard

## Running Containers
Clone Repo  
```git clone https://github.com/rzzldzzl/starlink-mon.git```  

Execute docker-compose  
```cd startlink-mon```  
```docker-compose up -d```

## Grafana Url
