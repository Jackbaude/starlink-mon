# starlink-mon
Docker Compose and Grafana configs to stand up:
* [starlink-grpc-tools](https://github.com/sparky8512/starlink-grpc-tools)
* InfluxDB
* Grafana
  * Starlink Dashboard

## Running Images
`docker-compose up -d`

## Grafana Url
http://localhost:3000
